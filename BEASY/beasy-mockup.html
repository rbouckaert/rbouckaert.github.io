Template

Using the standard template.

Data

There are 3 partitions (RSV2_1, RSV2_2 and RSV2_3 ) with 209, 210 and 210 sites respectively.

Site model

All partitions individually have a and substitution model[1] with kappa distributed (mean-log= and stdev-log=) and frequencies.

Relative substitution rates among all partitions are estimated.

Clock model

All partitions share a with clock rate distributed (mean-log= and stdev-log=).

Tree

There is a single tree with dated tips (in dates not ages) using tree prior with population size distributed.

References

This analysis is for BEAST [3].
